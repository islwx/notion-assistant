# notion-assistant
## about 
这是一个基于notion api的实现的一个chrome插件。  
用于做一些辅助性操作
## 安装
### 源码安装
chrome -> 更多工具 -> 扩展程序 -> 开发者模式 -> 加载已解压的扩展程序 -> 选中代码库的“src”文件夹
